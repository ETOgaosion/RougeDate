# Arknights Rouge with date options

As a universal framework, we seperate the date logics and the frameworks. We create a json format date arrangement file reading mechanism.

Put your arrangement json in [date_arrangement](./date_arrangement/) and deploy the app.

## Run locally

Install dependencies and start the website:

```powershell
npm install
npm run dev
```

Then open the local URL shown by Vite (usually `http://localhost:5173`).

Detailed setup notes are in [docs/install_and_view.md](./docs/install_and_view.md).
